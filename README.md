> Web application focused on the backend, use of nodeJs (express, cors, mongoose, multer, nodemon, sucrase), in addition to MongoDB for the database.

### 🏠 [Homepage](https://github.com/MariaE-duarda/devHouse#readme)

## Prerequisites

- npm >=5.5.0
- node >=9.3.0
- cors ^2.8.5
- express ^4.18.2
- mongoose ^6.8.1
- multer ^1.4.5-lts.1
- nodemon ^2.0.20
- sucrase ^3.29.0

## Install

```sh
yarn install
```
